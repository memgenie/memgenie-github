## What is Membean?

Membean is a vocabulary platform schools assign, usually for a set number of minutes per week. It teaches words through repeated exposure across different question formats — multiple choice, spelling, definition matching, a constellation image, a short passage using the word in context.

Membean doesn't only check whether an answer is right. It records how long a session takes, how consistently a student returns across the week, response time per question, and how accuracy moves over time. A teacher dashboard surfaces all of it.

## What MemGenie does

MemGenie is a Chrome extension that runs on Membean training sessions and quizzes. It reads the question as rendered on the page (not the underlying page source or API), and it can highlight or answer depending on which tier you're using.

## Why accuracy and timing are adjustable

A session that answers every question instantly and perfectly is a pattern a teacher dashboard can flag — nobody works that way. MemGenie's accuracy and timing controls exist so the pace and hit rate can be set by the user rather than fixed by the tool.

## FAQ

**Why do students look for Membean help?**
Membean is graded on time spent rather than mastery, so a required 15–45 minutes per week continues even after the words are known. Most people looking for a Membean tool are trying to reclaim that time, not avoid learning vocabulary.

**What kinds of Membean tools exist?**
Four broad categories: answer lookup tools that show the correct choice, full session automation, quiz/assessment solvers, and study aids that help you learn the list faster so sessions get shorter on their own.

**Can Membean detect automation?**
Membean records timing, accuracy, and answer patterns. Tools that answer instantly and perfectly with no variation produce a pattern that stands out. Pacing and occasional misses are far harder to distinguish from ordinary use.

**Are free Membean scripts safe to use?**
Free userscripts depend on Membean's page structure, which changes over time, so they tend to break without warning. Because they're unmaintained, a broken script can keep running long after Membean has changed how it reads sessions.

More detail on each of these: [full Membean guide hub](https://geniecheats.com/membean.html).
